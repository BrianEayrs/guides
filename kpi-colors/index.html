Funnel Tracking

Color-Code Your KPI Sheet

Put traffic-light colors on your KPI tracker so you can read performance in one glance. Green is on target, yellow is drifting, red needs attention. Copy the prompt below, paste it into Claude, and it either sets the colors up for you or hands you the click-by-click steps.

The prompt

Paste this whole block into Claude. It will ask for your sheet and confirm your columns, then apply the rules or give you exact steps.

Copy into Claude 
I want to color-code the KPI metrics in my Google Sheet so I can read performance at a glance. Set up traffic-light conditional formatting on each metric column:

- Green = on target
- Yellow = watch it, drifting
- Red = problem, needs attention

Use this exact rule set as the template. Every metric here is higher-is-better, so a bigger number is greener. Percentages are stored as fractions in the sheet (4% is saved as 0.04), so write the cutoffs in fraction form.

THREE-BAND metrics (red, yellow, green):

| Metric        | Red (low)   | Yellow (middle) | Green (good) |
|---------------|-------------|-----------------|--------------|
| CTR (All)     | < 0.03      | 0.03 to 0.06    | >= 0.06      |
| CTR (Link)    | < 0.01      | 0.01 to 0.03    | >= 0.03      |

TWO-BAND metrics (red, green):

| Metric                 | Red (low) | Green (good) |
|------------------------|-----------|--------------|
| Lead Conv Rate         | < 0.20    | >= 0.20      |
| Saw Offer Conv Rate    | < 0.20    | >= 0.20      |
| Front End Sale Conv Rate | < 0.03  | >= 0.03      |
| Book Call Rate         | < 0.70    | >= 0.70      |
| Show Rate              | < 0.80    | >= 0.80      |
| Close Rate             | < 0.20    | >= 0.20      |

Colors to use:

- Red:    #F4CCCC
- Yellow: #FFF2B2
- Green:  #CCEFCC

How to build each rule:

- Use a custom-formula rule so blank cells stay uncolored. For a column F rule, the green rule is: =AND($F2<>"", $F2>=0.06). Swap the column letter and cutoff per metric.
- Apply each rule from row 2 down to the bottom of the data.
- Order matters: Sheets stops at the first rule that matches. Put red first, then yellow, then green, so the bands don't overlap.

Now do this for me:

1. Ask me for my sheet (a link or the sheet ID) and which tab the metrics are on.
2. Ask me to confirm the column for each metric, since my layout may differ from the template above. If I track a metric where LOWER is better (like Cost Per Lead or CPM), flip the rule so the low end is green.
3. If you can edit my sheet through a tool, apply the rules directly. If you can't, give me the exact click-by-click steps in Google Sheets: Format > Conditional formatting > Custom formula is, then the formula, range, and fill color for each rule.
4. After you finish, tell me which columns you colored and the cutoffs you used, so I can spot-check.

If I don't have targets for a metric yet, pull my last 30 days, show me a typical good day versus a bad day, and suggest cutoffs before you apply anything.

The prompt carries its own instructions, so you do not need to add anything. Just paste and answer.

Why color beats a wall of numbers

A KPI sheet full of decimals hides the story. You scan it, everything looks fine, and you miss the one row that is quietly bleeding. Color fixes that. When your click rate goes red and your show rate stays green, you know in a second where the problem is and where it is not. You stop guessing and start fixing the right step.

The cutoffs are a starting point

The thresholds in the prompt are a sensible default for a webclass funnel. Your business may run at different numbers, and that is fine. When Claude asks, confirm your own targets, or let it pull your last 30 days and suggest cutoffs from what a good day and a bad day actually look like for you.

One thing to watch

Most funnel metrics are higher-is-better, so a bigger number gets greener. But a few, like cost per lead or cost per thousand views, are lower-is-better. For those, the rule flips: the low number should be green. The prompt already tells Claude to ask about this, so just point out any metric where cheaper or lower is the goal.

Set the colors once and your tracker starts telling you where to look. That is the whole point of tracking.
